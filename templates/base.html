📄 Recalbox Readme Cleaner

    
Finds and removes the multilingual readme files
       (_readme.txt, _leeme.txt, etc.)
       that Recalbox places in every system folder.

    

        
        
        
    

    
    


    


    
    
💿 CHD Verification Tool (chdman)

    
DAT Scanner and DAT Coverage use chdman
       (MAME's CD-image tool) to extract real track data from .chd files for hash
       verification against Redump DATs - currently supported for standard CD-track systems
       (e.g. Saturn, PSX, Sega/Mega CD); GD-ROM-based systems (Dreamcast) aren't supported yet.

    

        
        
        
    

    
    


    
    
🗄️ Result Cache

    
ROM hashes and DAT Scanner verify results are cached
       in a local SQLite database so repeat scans of unchanged folders skip the expensive work
       (CHD extraction, archive hashing) and return almost instantly. Clearing it just forces
       everything to be recomputed on the next scan - no ROM/DAT files are affected.
